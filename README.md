# Site static – starter
Fișiere:
- `index.html` – pagina principală
- `styles.css` – stiluri
- `script.js` – funcționalități minime
- `privacy.html` – exemplu pagină GDPR
- `img/logo.svg`, `img/favicon.svg`

## Publicare pe GitHub Pages
1. Creează un repo public pe GitHub (ex: `site-demo`).
2. Încarcă toate fișierele din acest folder în repo.
3. Settings → Pages → Source: **Deploy from a branch** → `main` → `/root` → Save.
4. După câteva zeci de secunde, vei avea un URL public.

## Domeniu propriu (opțional)
- Setează un CNAME la adresa GitHub Pages a repo-ului.
- Creează fișierul `CNAME` în rădăcină cu domeniul tău (ex: `exemplu.ro`).

## Notă
Formularul nu trimite nicăieri; pentru trimitere reală poți folosi servicii gen Formspree/Netlify Forms.
